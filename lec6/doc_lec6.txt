=== 사용자 함수 ===
1. 주의사항
함수 호출 전까지는 함수 안의 문장들은 수행 X
함수는 호출 되기 전에 먼저 만들어져야 함
함수의 정의 => def 함수이름(함수의 인수...):
2. 동작과정
반환을 하는지 안하는지 생각하기
def print_name():
def sum(a, b):
=> 반환값이 있을 경우 return 넣기
3. 만들어보자
  1) 문자열을 입력받음
  2) 문자열 내에 'skip'이라는 단어가 있으면 'rejected' 출력
  3) 'skip'이라는 단어가 없으면 그대로 출력
  4) 매 출력마다 '-'*10 출력해서 구분
  5) 사용자가 'quit'를 입력하여 끝낼 때까지 프로그램을 무한 반복
